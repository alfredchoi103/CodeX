# SLGP7007 Repository

This repository stores the course materials for SLGP7007 and the working notes for the Hong Kong urban flooding policy project.

## Contents

- `SLGP7007 Governance and Policy for Sustainability.docx`
- `SLGP7007-Brainstorming and discussion.docx`
- `SLGP7007-policy layer and effectness evaluation.docx`
- `capano-howlett-2020-the-knowns-and-unknowns-of-policy-instrument-analysis-policy-tools-and-the-current-research-agenda(1).pdf`
- `The Handbook of Environmental Policy Valuation.pdf`
- `threads/` - thread-by-thread working notes and decisions

## How to use this repo

- Keep course readings and source documents in the repository root or in a dedicated `sources/` folder if the project grows.
- Keep each discussion thread or workstream in `threads/` as a separate Markdown file.
- Use stable filenames with dates so we can track how the project evolved over time.

## Working convention

- `threads/` is for decision logs, outlines, and synthesis notes.
- `sources/` can be added later if we want to separate raw readings from working notes.
- Keep the main research line in sync with the presentation outline and the individual report outline.
